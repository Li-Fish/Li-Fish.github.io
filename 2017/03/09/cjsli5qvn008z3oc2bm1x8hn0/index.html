计算行列式（高斯消元？+Java+工具）

功能：

计算行列式并输出

用法：

首先输入行列式的阶数，然后以输入行列式内容。

例如：

输入：
4
1 2 -1 3
2 3 -1 2
-1 1 1 0
0 1 -2 1

输入：
18.0

实现：

好像是高斯消元，就是每一行乘一个系数减下去，化三角。
时间复杂度 O(n^3) .
